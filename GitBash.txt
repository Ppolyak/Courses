1) Посмотреть где я - pwd
2) Создать папку - mkdir papka 
3) Зайти в папку - cd papka
4) Создать 3 папки - mkdir 1 2 3 
5) Зайти в любоую папку - cd и путь к папке
6) Создать 5 файлов (3 txt, 2 json) - touch 1.txt 2.txt 3.txt 1.json 2.json
7) Создать 3 папки - mkdir 3 2 1
8. Вывести список содержимого папки - ls
9) + Открыть любой txt файл - vim
10) + написать туда что-нибудь, любой текст.
11) + сохранить и выйти. - :wq
12) Выйти из папки на уровень выше - cd ..

13) переместить любые 2 файла, которые вы создали, в любую другую папку. - mv 
14) скопировать любые 2 файла, которые вы создали, в любую другую папку. - cp 
15) Найти файл по имени - find / -name (имя файла)
16) просмотреть содержимое в реальном времени (команда grep) изучите как она работает.
17) вывести несколько первых строк из текстового файла - head -кол-во_строк имя_файла
18) вывести несколько последних строк из текстового файла - tail -кол-во_строк имя_файла
19) просмотреть содержимое длинного файла (команда less) изучите как она работает.
20) вывести дату и время - date
